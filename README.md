# Heart-Disease-Prediction
Heart Disease Prediction using Machine Learning (Logistic Regression, Random Forest, etc.) to classify patients based on medical attributes.

# Overview
This project predicts the likelihood of heart disease in patients using machine learning algorithms. It uses medical attributes such as age, cholesterol level, blood pressure, and more to classify whether a person has heart disease.

# Objective
To build a predictive model that can accurately identify patients at risk of heart disease based on clinical data.

# Dataset
Source: (Kaggle Custom dataset)
https://www.kaggle.com/datasets/mdnazmussadat1/heart-disease

# Technologies Used
Python 
NumPy
Pandas
Matplotlib / Seaborn
Scikit-learn

# Machine Learning Models

This Model Trained and evaluated four classification models for heart disease prediction 
Logistic Regression: A linear model for binary classification, estimating class probabilities.
Support Vector Machine (SVM): A powerful model finding optimal hyperplanes, often using an RBF kernel.
Random Forest: An ensemble method using multiple decision trees for robust classification.
XGBoost: An optimized gradient boosting framework known for efficiency and performance.
Models were evaluated using Accuracy, ROC-AUC, F1 Score, and Cross-Validation Accuracy.

# Result
The analysis trained four classification models, with Logistic Regression emerging as the best performer based on ROC-AUC, achieving an AUC of 0.9948 and an accuracy of 0.9650.
